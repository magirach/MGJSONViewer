## Installation

MGJSONViewer is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'MGJSONViewer'
```

## Use

```swift
    MGJSONViewer()
    tbl.loadData(fileName: "json", ext: "txt")
```
